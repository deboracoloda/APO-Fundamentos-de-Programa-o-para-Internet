# Gerador de Currículo (PHP)

Projeto simples para a disciplina **Fundamentos de Programação para Internet**.

## Requisitos
- PHP (XAMPP) — colocar a pasta `curriculo/` em `htdocs/`

## Como executar
1. Copie esta pasta para `C:\xampp\htdocs\curriculo` (Windows) ou `/Applications/XAMPP/htdocs/curriculo` (macOS).
2. Inicie o Apache no XAMPP.
3. Acesse `http://localhost/curriculo/`.
4. Preencha o formulário e clique em **Gerar Currículo**.
5. Na tela do currículo, use **Imprimir / Baixar** (usa `window.print()`).

## Tecnologias
- **Frontend:** HTML + Tailwind CSS via **jsDelivr** (`<script src="https://cdn.jsdelivr.net/npm/@tailwindcss/browser@4"></script>`) + JavaScript (campos dinâmicos, cálculo de idade)
- **Backend:** PHP (montagem e formatação do currículo)
